# AI Usage Documentation

## Project Overview

This project uses GitHub Copilot as a tutoring assistant to guide learning of front-end and back-end development concepts, following curriculum requirements step by step.

---

## Active Mode for This Project: TUTOR MODE

### TUTOR MODE Instruction

```
I'm learning front-end/backend development and want you to be my tutor.
Please guide me step by step without giving me finished solutions:

1. Ask me guiding questions to help me think through problems
2. Explain concepts and the "why" behind approaches
3. Give me hints and pointers on where to look
4. Review my code and suggest improvements with explanations
5. Break down complex problems into smaller steps

Don't just give me the answer - help me understand and figure it out myself.
```

**Purpose:** Learning to understand Express.js middleware concepts through practical implementation, following curriculum guidelines.

---

## Alternative Mode: PROOF-READER MODE

### PROOF-READER MODE Instruction

```
Please review my code as a tutor would. For each piece of code I share:

1. Point out what works well
2. Identify any bugs, issues, or improvements needed
3. Explain why something might be problematic
4. Suggest better approaches with explanations
5. Ask clarifying questions about my intent if needed

Help me understand the issues and how to fix them, not just fix it for me.
```

**When to use:** For code review and quality improvement across different projects.

---

## How AI is Being Used in This Project

- **Learning Guidance:** Explaining middleware concepts and Express patterns
- **Step-by-Step Support:** Breaking down tasks into manageable pieces
- **Code Review:** Reviewing student-written code for learning purposes
- **Concept Clarification:** Explaining the "why" behind best practices
- **Not Used For:** Providing finished solutions without understanding

---

## Student Responsibility

As the learner, I am responsible for:

- Writing and understanding all code I submit
- Not copy-pasting solutions without comprehension
- Asking clarifying questions when concepts are unclear
- Taking time to think through problems before asking for help
- Using AI as a tutor, not as a substitute for learning
